# 💸 Expense Tracker

A lightweight expense tracker built with vanilla HTML, CSS, and JavaScript.

## Features

- Add transactions with a description, amount, category, and date
- Automatically calculates and displays:
  - Total balance
  - Total spendings
  - Total earnings
- Delete transactions with live summary updates

## How to Use

1. Enter a description and amount
2. Select whether it's a **Spending** or **Earning**
3. Click **Add Transaction**
4. Hit **✕** on any transaction to delete it

## Tech Stack

- HTML5
- CSS3 (Flexbox)
- Vanilla JavaScript (DOM manipulation, event delegation)
